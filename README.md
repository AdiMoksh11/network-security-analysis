# Network Traffic Analysis for Real-Time Detection of Cyber Attacks

A deep learning–based system for detecting cyber attacks from network traffic in real time.

---

## 📌 Project Overview

This project aims to build an intelligent Intrusion Detection System (IDS) capable of identifying malicious network traffic using machine learning and deep learning techniques.

The system will:

- Capture and analyze network traffic
- Preprocess and extract useful features
- Train machine learning and deep learning models
- Detect cyber attacks in real time
- Visualize predictions through an interactive dashboard

## ⚙️ Setup

### Windows

```bash
py -3.10 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### Linux/macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
